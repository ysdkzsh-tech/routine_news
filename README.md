# routine_news

瀬戸市尾張旭市在宅医療クリニック（2027年4月開業予定）向けの週次情報収集・PDCA提案ルーティン用リポジトリ。

クラウドエージェントが毎週、Web上の関連情報を収集・要約し、`reports/YYYY-MM-DD.md` に計画・実行・評価（PDCA）の提案をコミットする。

- `CONTEXT.md` — クリニックの背景情報（エージェントが毎回参照する）
- `reports/` — 週次レポートの保存先

## ルーティンの実行

- `schedule`スキルで作成したクラウドエージェントのcronジョブが、毎週月曜9:00 JSTに自動起動する。
- 実行時の詳細手順は [`ROUTINE.md`](ROUTINE.md) に記載されている。
- レポート完成後は自動で `origin/main` にpushされ、完了時にプッシュ通知が送られる。
